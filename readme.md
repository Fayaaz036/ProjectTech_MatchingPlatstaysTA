# Matching Playstation App

## Idee Generatie
Dit word een platform waar je je kan inschrijven voor Playstation games. 
Hier maak je een account aan, en word er aan de hand van je PSN geladen welke games je hebt gespeeld. 

## Job stories 
1. Als gebruiker wil ik games zien die mij worden aangeraden op basis van wat ik gespeeld heb op mijn Playstation account.

2. Als gebruiker wil ik games op kunnen slaan om die later uit te checken.

3. Als gebruiker wil ik met anderen die die games spelen contacten, en misschien samen spelen.

4. Als gebruiker wil ik mijn achievements voor een game showen voor anderen. 

### Functie die ik ga uitwerken
Ik ga functie 4 uit werken. Ik ga de games laten zien die je hebt gespeeld, en hierbij ga ik je behaalde achievements laten zien per game.

## Requirements
* Api van PSN 
* Database met games als fallback om te selecteren
